## What it distills

| Layer | Description |
|---|---|
| **Lexical Signature** | Preferred words, forbidden words, unique slang. |
| **Syntactic Rhythm** | Sentence length, clause usage, punctuation habits. |
| **Imagery Palette** | Sensory focus (e.g., Hemingway's concrete nature vs Marquez's magical metaphor). |
| **Writing DNA** | Openings, conclusions, narrative perspective. |

---

## How it works

**1. Multi-dimensional Research**: 6 Agents run in parallel to gather data (Books, Interviews, Letters, Critic Perspectives).
**2. Triple Distillation**:
   - **Semantic Fingerprinting**: Extract unique word combinations.
   - **Rhythm Anchoring**: Deconstruct signature sentence structures.
   - **Imagery Library**: Build a sensory description dictionary.
**3. Style Injection**: Inject these "fingerprints" into your original text for alignment and rewriting.
